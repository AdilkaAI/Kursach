##  Основные возможности

 **Регистрация и авторизация** 
 **Страница экспертов** с подробной информацией и статусом (Онлайн / Занят)
 **Интерактивный чат** с реальными ответами от ИИШКИ (Gemini)
 **История чатов** ("Мои чаты")
 **Редактирование профиля** и настройки
 **База знаний** (часто задаваемые вопросы)
 **Адаптивный дизайн** + поддержка тёмной темы
 **Завершение чата** с автоматическим освобождением эксперта

##  Технологический стек

### Frontend
 React + Vite
 React Router DOM
 Axios

### Backend
 Node.js + Express
 MongoDB + Mongoose
 JWT (авторизация)
 Google Gemini AI (для умных ответов экспертов)
